Please read this Privacy Policy carefully before using this app.

Article 1 (User Information Items Sent from This App)

(1) User Information Automatically Obtained in This App

We obtain information about the user's application usage and device information (OS information, etc.) when the user uses this app.

We do not collect information that can identify individual users when they use this app.

(2) User Information Obtained Through Social Features

When using the social features of this app, we collect the following information:

  • Age (information manually entered by the user)
  • Region (information manually entered by the user)
  • Country name (information automatically obtained by the app)
  • Icon and color (information manually set by the user)
  • Bucket list post content (when the user chooses to make it public)
  • Like information (information about which user liked which post)
  • Report information (information about which user reported which post)

Note that this information does not identify individuals. Age and region are information that users voluntarily enter, and accuracy is not guaranteed.

Article 2 (Purpose of Use)

We will use the information collected from users, including user information, device information, and other information related to the use of this app, for the purpose of operating, developing, and improving this app, responding to inquiries, improving services, and developing new products, and will handle it appropriately in accordance with this Privacy Policy.

Information obtained through social features is used for the following purposes:

  • Bucket list post content: Display to other users, provision of like and report features
  • Age, region, country name, and icon information: Display of user profiles
  • Like information: Provision of like features and creation of statistical information
  • Report information: Management of inappropriate content and provision of automatic filtering features

Article 2.2 (Data Retention Period for Social Features)

The data retention period for information obtained through social features is as follows:

  • Public bucket list post content: Retained until deleted by the poster
  • Even when an account is deleted, posts remain unless manually deleted by the poster
  • Like information and report information: Automatically deleted when the account is deleted
  • Age, region, country name, and icon information: Automatically deleted when the account is deleted

Article 3 (Prohibition of Disclosure/Provision of Personal Information to Third Parties)

We will properly manage the personal information entrusted to us by customers and will not disclose personal information to third parties except in the following cases.

  • When the customer has given consent
  • When disclosure is required by law

Article 4 (Change of Purpose of Use)

We will change the purpose of use of personal information only when it is reasonably recognized that the purpose of use after the change is related to the purpose of use before the change.

When the purpose of use is changed, we will notify users of the changed purpose by our prescribed method or publish it on this website.

Flag Memory AwaPop Privacy Policy

This Privacy Policy defines the handling of user information sent from the application service "Flag Memory AwaPop" (hereinafter referred to as "this app") provided by takekapp1990 (hereinafter referred to as "we"), and applies to all customers (hereinafter referred to as "users") who use this app.

Please read this Privacy Policy carefully before using this app.

By using the app, users are deemed to have agreed to this Privacy Policy.

Article 1 (User Information Items Sent from This App)

(1) User Information Automatically Obtained in This App

We obtain information about the user's application usage and device information (OS information, etc.) when the user uses this app.

We do not collect information that can identify individual users when they use this app.

User Statistics Data for Rankings

This app collects and stores the following information to provide ranking features.

  • User ID (device ID)
  • Display name (player name)
  • Avatar image URL
  • User level, experience points, number of avatars owned
  • Last update date and time
  • Language code, country code, platform information

This information is stored in Firestore and is used for displaying rankings, recording user game progress, and filtering rankings by category. It is retained until the user uninstalls the app.

Game Result Data (for Rankings)

This app collects and stores the following information to provide ranking features.

  • Game result ID
  • User ID (device ID)
  • Score
  • Game settings information (game mode, category, difficulty, etc.)
  • Play date and time
  • Other game statistics

This information is stored in Firestore and is used for displaying monthly and overall rankings, and recording and comparing user performance. It is retained permanently.

Usage Data (Analytics)

This app collects and stores daily and monthly usage data (app usage status, frequency of feature usage, etc.) to improve the app and develop features.

This information is stored in Firestore and is used for app improvement and feature development, and usage analysis. It is retained permanently (for aggregation and analysis).

Device Information

This app collects and stores device ID (unique identifier) and platform information (iOS, Android, Web) to identify users and manage ranking data.

This information is stored in local storage and Firestore, and is used for user identification, ranking data management, and app feature provision. It is retained until the user uninstalls the app.

(2) Contact Information Regarding This App

When users make inquiries about this app, we may ask for their email address and other information related to the inquiry as necessary.

(3) Permissions Used

Local Folder Access (path_provider)

This app accesses the app's sandbox (iOS) or internal storage (Android) to save app data (settings, game data, etc.) and temporary files.

We only access the app's sandbox and do not access other apps' or users' files. No additional permission settings are required.

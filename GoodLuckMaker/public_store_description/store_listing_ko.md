# ストア掲載文（韓国語）

アプリ名
절대 행복 일기

短い説明文（iOS用 / 100文字以内）
귀여운 캐릭터와 함께하는 한마디 일기 앱

短い説明文（Android用 / 80文字以内）
귀여운 캐릭터와 함께하는 한마디 일기와 오늘의 운세

キーワード（100文字以内、カンマ区切り）
일기,한마디 일기,한줄 일기,자기돌봄,셀프케어,감정기록,습관 형성,캐릭터 육성,미니게임,마음챙김,힐링,기분관리,매일기록

プロモーション用テキスト（iOS用 / 170文字以内）
매일 스스로를 칭찬해 보세요. 짧은 일기, 오늘의 운세, 캐릭터 돌보기를 통해 부담 없이 꾸준한 루틴을 만들 수 있어요.

詳細の長い説明文
절대 행복 일기는 매일을 조금 더 다정하게 기록하는 한마디 일기 앱입니다.
더 잘해야 한다는 압박 대신, 스스로를 칭찬하는 습관을 만들 수 있도록 설계했어요.
짧은 기록만으로도 충분히 이어갈 수 있고, 캐릭터와 함께라서 더 즐겁습니다.

주요 기능
・매일 한마디 일기
오늘의 기분이나 해낸 일을 짧게 남겨 보세요.

・도움이 되는 알림 기능
일기 루틴을 놓치지 않도록 부드러운 알림으로 도와줍니다.

・매일 긍정 메시지
아침에 짧은 긍정 메시지를 읽고 기분 좋게 하루를 시작하세요.

・캐릭터/배경 커스터마이즈
취향에 맞게 꾸며서 매일 열고 싶은 앱으로 만들 수 있어요.

・귀여운 배지 시스템
꾸준히 사용할수록 배지가 쌓여서 성장 실감이 커집니다.

・사운드와 보이스
소리 기능을 켜면 일상 기록이 더 생생하고 즐거워져요.

이런 분께 추천해요
・가벼운 일기 습관을 만들고 싶은 분
・자기돌봄과 감정기록을 꾸준히 이어가고 싶은 분
・귀여운 캐릭터와 함께 동기부여를 받고 싶은 분

추천 사용 루틴
・아침: 긍정 메시지 읽기
・저녁: 한마디 일기 작성
・틈날 때: 미니게임과 캐릭터 돌보기

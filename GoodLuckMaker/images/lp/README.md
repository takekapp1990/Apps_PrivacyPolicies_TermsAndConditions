# LP Image Layout

Canonical structure for multilingual LP assets.

- `shared/backgrounds/`: language-agnostic background textures
- `shared/characters/f01/`: cross-locale character cutouts (f01 set)
- `platforms/android/locales/<locale>/feature/featureGraphic.png`: locale-specific feature graphic
- `platforms/android/locales/<locale>/screenshots/phone/*.png`: locale-specific phone screenshots
- `platforms/android/locales/<locale>/icon/icon.png`: optional locale icon if provided

Current locales copied:
- `ja-JP`, `en-US`, `es-ES`, `id`, `ko-KR`, `pt-BR`, `zh-CN`

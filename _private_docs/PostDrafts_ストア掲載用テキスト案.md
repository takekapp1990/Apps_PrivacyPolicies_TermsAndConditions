# Post Drafts ストア掲載用テキスト案

## 📱 アプリ概要
**Post Drafts** - SNS投稿をもっと楽に、もっと効果的に

SNS投稿の下書き管理を効率化するアプリ。文字数カウント、ハッシュタグ管理、OGP確認、翻訳機能で完璧な投稿を実現。

Post Drafts - SNS投稿用メモアプリで、もっと楽に、もっと効果的に！

SNS投稿でこんな悩みはありませんか？
• 文字数制限を超えないか心配
• ハッシュタグを毎回手で打っていて面倒
• OGP画像がうまく取得できているか不安
• Xの下書きをもっと効率的に行いたい

Post Draftsは、SNS投稿専用のメモアプリ。あなたのSNS投稿の悩みを解決します！

## 主要機能 ##
• メモ作成・管理 - SNS投稿用のメモを効率的に作成・編集・削除
• リアルタイム文字数カウント - Xの文字数制限に合わせた計算で安心投稿
• ハッシュタグ管理 - よく使うハッシュタグを登録してワンタップで挿入
• OGP確認機能 - URLを入力するだけでOGP画像をプレビュー
• 翻訳機能 - Google翻訳で日本語と英語の両方に対応
• タグ機能 - メモをタグで分類して整理整頓
• 検索・フィルタリング機能 - 過去のメモを簡単に検索・振り返り
• コピー機能 - 既存メモをコピーして連続投稿を効率化
• データ管理・引き継ぎ機能 - エクスポート・インポート対応
